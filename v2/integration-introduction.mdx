This tutorial guides you through a basic interaction with the Chargily Pay API. You'll make your first API call to generate a new [Checkout](/v2/api-reference/checkouts/checkout-object "Checkout") and retrieve the payment URL. For additional details regarding other API objects, please refer to the [API reference](/v2/api-reference/introduction "API reference").
